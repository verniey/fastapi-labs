## TECHNOLOGIES

- Python 3.11.4
- pip-tools
- FastAPI
- Postgres
- SQLAlchemy
- Uvicorn

## Local Development

1. Clone this repo and `cd` into repo directory
2. python3 -m venv venv
3. Create vertual enviroment `source venv/bin/activate`
4. python3 -m pip install pip-tools
5. pip-compile and pip-sync
